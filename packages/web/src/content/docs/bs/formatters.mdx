## Kako radi

Kada SumoCode piŇ°e ili ureńĎuje datoteku, on:

1. Provjerava ekstenziju datoteke prema svim omoguńáenim formaterima.
2. Pokreńáe odgovarajuńáu naredbu za formatiranje na datoteci.
3. Automatski primjenjuje promjene formatiranja.
   Ovaj proces se dogańĎa u pozadini, osiguravajuńái da se vaŇ°i stilovi koda odrŇĺavaju bez ikakvih ruńćnih koraka.

## Konfiguracija

MoŇĺete prilagoditi formatere kroz `formatter` odjeljak u vaŇ°oj SumoCode konfiguraciji.

```json title="sumocode.json"
{
  "$schema": "https://opencode.ai/config.json",
  "formatter": {}
}
```

Svaka konfiguracija formatera podrŇĺava sljedeńáe:
| Svojstvo | Vrsta | Opis
|------------- | -------- | ------------------------------------------------------- |
| `disabled` | boolean | Postavite ovo na `true` da onemoguńáite formater |
| `command` | string[] | Naredba za pokretanje za formatiranje |
| `environment` | objekt | Varijable okruŇĺenja koje treba postaviti prilikom pokretanja formatera |
| `extensions` | string[] | Ekstenzije datoteka koje ovaj formater treba da obrańĎuje |
Pogledajmo neke primjere.

---

### Onemoguńáavanje formatera

Da onemoguńáite **sve** formatere globalno, postavite `formatter` na `false`:

```json title="sumocode.json" {3}
{
  "$schema": "https://opencode.ai/config.json",
  "formatter": false
}
```

Da onemoguńáite **specifińćni** formater, postavite `disabled` na `true`:

```json title="sumocode.json" {5}
{
  "$schema": "https://opencode.ai/config.json",
  "formatter": {
    "prettier": {
      "disabled": true
    }
  }
}
```

---

### PrilagońĎeni formateri

MoŇĺete nadjańćati ugrańĎene formatere ili dodati nove navońĎenjem naredbe, varijabli okruŇĺenja i ekstenzija datoteke:

```json title="sumocode.json" {4-14}
{
  "$schema": "https://opencode.ai/config.json",
  "formatter": {
    "prettier": {
      "command": ["npx", "prettier", "--write", "$FILE"],
      "environment": {
        "NODE_ENV": "development"
      },
      "extensions": [".js", ".ts", ".jsx", ".tsx"]
    },
    "custom-markdown-formatter": {
      "command": ["deno", "fmt", "$FILE"],
      "extensions": [".md"]
    }
  }
}
```

**`$FILE` ńćuvar mjesta** u naredbi ńáe biti zamijenjen putanjom do datoteke koja se formatira.
